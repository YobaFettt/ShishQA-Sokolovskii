HTTP - протокол верхнего уровня передачи данных в интернете, изначально предназначался для передачи гипертекстовых документов, в настоящее время используется для передачи произвольных данных;
HTTPS - расширение протокола HTTP, в который включена поддержка шифрования (криптографические протоколы TLS, ранее - SSL). Изменен TCP порт с 80 на 443.

Версии HTTP 
- HTTP 1.1 (до 1999 года версии 0.9 и 1.0);
- HTTP/2.

Отличия HTTP 1.1 от HTTP/2:
- HTTP/2 первый бинарный протокол, сервера могут доставлять информацию, которую клиент еще не запросил;
- мультеплексирование запросов и ответов (решает проблему блокировки начала строки);
- можно сжимать HTTP заголовки;
- снижена скорость загрузки страницы.

DNS - система доменных имен, связывает наименование сайта (то, что вводится в адрессную строку), и его фактический адрес (IP адрес). Другими словами DNS преобразует имя ресурса в адресной строке в IP адрес и передает устройству. Далее сервер, находящийся по данному адресу, обрабатывает запрос и присылает информацию для открытия страницы.
