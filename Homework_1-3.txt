Тест-план куринного яйца
#
Introduction
#
Тестирование куринных яиц, проверка продукта на свежесть и возможность приготовления для последующего употребления конечным потребителем.
#
#
#
Scope of work
#
#
#
Будет производится тестирование следующих составляющих:
а) внешние (скорлупа);
б) внутренние (желток и белок).
#
Что нужно протестировать:
- внешний вид на отсутствие трещин, загрязнений;
- свежесть продукта путем погружения в воду, а также проверки даты производства, либо при помощи овоскопа;
- состояние яйца (вареное либо сваренное вкрутую);
- возможность приготовления на различных варочных поверхностях (газовая, электро, либо индуктивная);
- возможность приготовления различных блюд;
- возможность употребления в пищу после приготовления;
- возможность отделения желтка от белка разными способами.
#
Что не нужно тестировать:
- массу продукта в сыром и приготовленном виде;
- прочность скорлупы при ударных нагрузках.
#
#
#
Для тестирования необходимо:
- элетрическая, газовая и индуктивная варочные поверхности;
- ёмкость с водой;
- овоскоп;
- нож кухонный;
- молоко;
- сковорода антипригарная;
- сепаратор.
#
#
#
Test levels
#
#
1. Smoke test
Будет производится визуальное тестирование нескольких экземпляров для определения возможности проведения дальнейших тестов. При соответсвии экземпляры отправляются в отбраковку.
2. Critical path test
Определение возможности приготовления в различных условиях. Тесты проводятся после успешного прохождения Smoke тестирования.
#
#
Smoke test
#
#
1. Внешний осмотр на наличие трещин и загрязнений
Steps:
- взять яйцо;
- произвести осмотр.
Expected Result:
- яйцо без трещин и загрязнений.
#
#
2. Проверка внутреннего состояния яйца овоскопом
Steps:
- взять яйцо;
- включить лампу овоскопа;
- положить яйцо в ячейку овоскопа.
Expected Result:
- желток и белок имеют однородную структуру;
- отсутствуют потемнения.
#
#
3. Проверка свежести яйца путем сверки даты производства
Steps:
- взять яйцо;
- найти маркировку даты;
- сравнить с текущей.
Expected Result:
- дата производства отличается от текущей даты не более срока годности, указанного на коробке продукта.
#
#
4. Проверка свежести путем погружения в воду
Steps:
- налить в глубокую ёмкость воду;
- взять яйцо;
- положить в миску с водой.
Expected Result:
- яйцо находится на дне ёмкости.
#
#
5. Определение состояния яйца
Steps:
- взять яйцо;
- аккуратно раскрутить на плоской поверхности.
Expected Result:
- если яйцо крутится неравномерно и медленно --> сырое, проводим тесты по приготовлению;
- если яйцо крутится быстро и равномерно --> сваренное вкрутую, дальнейшие тесты проводить не целесообразно.
#
#
Critical path test
#
#
1. Приготовление яйца вкрутую в ёмкости с водой на электроплите
Steps:
- взять яйцо;
- налить в ёмкость воду;
- аккуратно положить яйцо в ёмкость;
- включить электроплиту;
- довести воду до кипения;
- варить в течении 8 минут;
- достать из горячей воды и охладить;
- разбить скорлупу и разрезать ножем.
Expected Result:
- желток твердный;
- желток и белок имеют однородный цвет.
#
#
2. Приготовление яйца вкрутую в ёмкости с водой на газовой плите
Steps:
- взять яйцо;
- налить в ёмкость воду;
- аккуратно положить яйцо в ёмкость;
- включить газ;
- аккуратно поджечь газ у горелки;
- довести воду до кипения;
- варить в течении 8 минут;
- достать из горячей воды и охладить;
- разбить скорлупу и разрезать ножем.
Expected Result:
- желток твердный;
- желток и белок имеют однородный цвет.
#
#
3. Приготовление яйца вкрутую в ёмкости с водой на индукционной плите
Steps:
- взять яйцо;
- налить в ёмкость воду;
- аккуратно положить яйцо в ёмкость;
- довести воду до кипения;
- варить в течении 8 минут;
- достать из горячей воды и охладить;
- разбить скорлупу и разрезать ножем.
Expected Result:
- желток твердный;
- желток и белок имеют однородный цвет.
#
#
4. Проверка возможности употребления в пищу яйца, сваренного  вкрутую
Steps:
- взять сваренное вкрутую яйцо;
- удалить скорлупу;
- разрезать яйцо ножем.
Expected Result:
- отсутствуют неприятные запахи;
- структура и цвет однородные, можно употреблять в пищую.
#
#
5. Приготовление яйца всмятку
Steps:
- взять яйцо;
- налить в ёмкость воду;
- аккуратно положить яйцо в ёмкость;
- включить варочную поверхность;
- довести воду до кипения;
- варить в течении 6 минут;
- достать из горячей воды и охладить;
- разбить скорлупу и разрезать ножем.
Expected Result:
- желток мягкий, вытекающий;
- желток и белок имеют однородный цвет.
#
#
6. Тестирование яйца на аккуратное разбитие скорлупы
Steps:
- взять яйцо;
- аккуратно разбить скорлупу ножем либо о край ёмкости для приготовления;
Expected Result:
- яйцо разбито, скорлупа удалена.
#
#
7. Приготовление омлета из яйца
Steps:
- взять бутылку молока;
- налить молоко в стакан;
- в ёмкость с разбитым яйцом вылить молоко;
- взбить полученную смесь;
- включить варочную поверхность;
- поставить ёмкость с полученной смесью на варочную поверхность;
- довести до кипения;
- снять с варочной поверхности через 5-7 минут.
Expected Result:
- омлет готов, имеет однородную структуру и светлый оттенок.
#
#
8. Приготовление яичницы
Steps:
- взять яйцо;
- поставить сковороду на варочную поверхность;
- на нагретую сковороду аккуратно разбить яйцо;
- по готовности снять сковороду с варочной поверхности;
- переложить готовую яичницу на тарелку.
Expected Result:
- отсутвуют посторонние запахи;
- блюдо имеет однородную структуру и цвет.
#
#
9. Возможность отделения желтка от белка традиционным способом
Steps:
- взять яйцо;
- ножем подбить скорлупу по центру яйца;
- аккуратно разломить яйцо на две половинки;
- над ёмкостью переливать содержимое яйца их ожной половины скорлупы в другую пока весь белок не стечет в ёмкость;
- перелить желток в другую ёмкость.
Expected Result:
- желток и белок отделены и находятся в разных ёмкостях.
#
#
10. Возможность отделения желтка от белка сепаратором
Steps:
- взять яйцо;
- ножем подбить скорлупу по центру яйца;
- аккуратно разломить яйцо на две половинки;
- взять сепаратор;
- перелить белок через сепаратор в ёмкость;
- желток перелить из сепаратора в другую ёмкость.
Expected Result:
- желток и белок отделены и находятся в разных ёмкостях.
